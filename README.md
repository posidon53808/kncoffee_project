# kncoffee_project
<h1>咖啡相關產品購物車系統</h1>
<hr>
<h3>專案內容</h3>
<p>專案的功能：為消費端建置會員登入功能，消費者能夠瀏覽商家的商品、菜單、資訊以及得知最新的消息及活動</p>
<p>消費者能夠選購商家產品，以及確認訂單，確認訂單後訂單會送到後台</p>
 

 <h3>使用技術</h3>

<li>SS2H框架</li>
<li>HTML、CSS</li>
